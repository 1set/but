Including:

* Python 3.6.8
* Node.js v12.7.0 & npm 6.10.0
* Go 1.12.7
* Zsh & Oh My Zsh
* curl, wget, rsync, netcat, lynx ...
* vim, git, zip, 7z, tmux, htop ...
